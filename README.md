# Neumorphism Profile Card UI Design using only HTML & CSS

Um projeto simples de **profile card com efeito neumorphism** usando apenas HTML e CSS.

## Sobre

Este projeto demonstra como criar um **card de perfil moderno com estilo neumorphism**, utilizando sombras suaves para um visual elegante.

## Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## Funcionalidades

- Card de perfil estilizado
- Efeito neumorphism
- Design moderno e minimalista

## Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
